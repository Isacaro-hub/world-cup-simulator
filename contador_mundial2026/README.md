# World Cup Simulator

Interactive World Cup simulator where users can predict and save match results.

## Features

- Add match results
- Save data in local storage
- Reset tournament data
- Dynamic standings table
- Responsive design
- Persistent data after page reload

## Technologies Used

- React
- Vite
- JavaScript

## Future Improvements

- Knockout stage simulation
- Team statistics
- Dark mode
- Match history
- Group auto-calculation

## Installation

```bash
npm install
npm run dev
```
